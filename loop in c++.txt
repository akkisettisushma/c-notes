                     loop in c++

loop in c++ are used to execute a block of code repeatedly until a certain condition is met .they help in manage in repetitive tasks.

1.for loop: used when the number of iterations is known before head.
 
2.while loop: used when number of iterations is not known and depend on condition.

3.do-while loop: similar to the while loop. but the guarantees that the block of code will be executed at least once before checking the               conditions



                  ----------------------------------------------------------------------------------------------------------

                             code on FOR loop

// printing numbers from 1 to 5 using for loop
#include <iostream>
using namespace std;

int main()
{
  for( int i = 1 ; i<=5 ;i++)
    {
        cout<<i<<endl;
    }
 
    return 0;
}

//printing numbers from 1 to 5 using while loop
#include <iostream>
using namespace std;
int main()
{
    int i=1;
 
    while (i<=5)
    {
        cout<<i<<endl;
        i++;
    }

    return 0;
}
